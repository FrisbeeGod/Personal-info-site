Interactive App

TEXTRIS is a game that I designed with my partner Felix Carrillo. TEXTRIS has the same elements as the game TETRIS however instead of eliminating blocks when completeing a row, blocks are eliminated when a word is formed. This means that each square of the normal TETRIS blocks will have a letter. The plan is to fully finish the project and publish it as an online game that will be fun and educational, by that we mean a game for kids to play when they are bored in class.
